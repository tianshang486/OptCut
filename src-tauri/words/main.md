# 使用教程

## 功能介绍

`所有界面中默认双击和Ctrl+C是复制,右键和ESC是关闭,使用时候可以多试试`

#### 截图OCR翻译:目前快捷键比较少后面增加

`目前最高只能开启20个截图窗口,底层限制了,后期想办法解决,如果出现窗口已满,右键点击菜单栏,关闭所有截图窗口即可解决。`

1. 截图: 默认ALT+CTRL+Q
2. 截图复制: 默认ALT+CTRL+W
3. 截图OCR识别: 默认ALT+CTRL+E
4. OCR后翻译: 右键点击图片会出现菜单,默认是自动->英语,可以点击翻译设置切换,双击翻译设置中间箭头可以关闭他。
5. 按空格键会开启绘画模式,注意OCR后就无法绘图了,目前绘画比较简单,未来可期

![image-20260119200836152](main.assets/image-20260119200836152.png#60%)

#### 划词翻译

- 默认使用ALT+E,目前不支持修改,后面会增加可修改
- 有普通模式和划词模式,看你需求来,划词模式会使用jieba分词在去重进行翻译
- 目前有个bug,鼠标离开3s后有大概率不会自动关闭,还在想办法解决,目前使用ESC可以关闭
